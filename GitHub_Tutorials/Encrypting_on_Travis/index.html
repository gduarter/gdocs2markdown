    • Note, if you’ve changed names at any point you might have set the repository with the -R option, for example

    • Run this command inside the git repository.

encrypting oe_license.txt for MobleyLab/off_nitrogens

storing result as oe_license.txt.enc

storing secure env variables for decryption

Please add the following to your build script (before_install stage in your .travis.yml, for instance):

openssl aes-256-cbc -K $encrypted_ff9126da2177_key -iv $encrypted_ff9126da2177_iv -in oe_license.txt.enc -out oe_license.txt -d

Pro Tip: You can add it automatically by running with --add.

Make sure to add oe_license.txt.enc to the git repository.

Make sure not to add oe_license.txt to the git repository.

Commit all changes to your .travis.yml

When things don’t work try adding the options —debug and —explode to the encrypt line so the first command would be

travis encrypt-file oe_license.txt -R MobleyLab/chemper --debug --explode
